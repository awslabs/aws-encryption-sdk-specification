GitHub Personal Access Token

Enter a GitHub personal access token to increase API rate limits. The token is stored only for this browser session.

Required permissions: No specific scopes needed for public repositories.

How to Create a Personal Access Token

  1. Go to GitHub Settings → Developer settings → Personal access tokens
  2. Click "Generate new token (classic)"
  3. Give it a descriptive name (e.g., "GitHub Dashboard")
  4. Set an expiration date (recommended: 90 days)
  5. For public repositories, no scopes are required
  6. Click "Generate token" and copy the token
  7. Paste it in the field above and click "Save Token"
Security Note: The token is stored only in your browser's session and will be cleared when you close this tab. It is never sent to any server except GitHub's API.
